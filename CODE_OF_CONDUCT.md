# Code of Conduct

## Our pledge

The GSM project is committed to a welcoming, harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our standard

The GSM project follows the **CNCF Community Code of Conduct**. By participating — contributing code or specification text, filing issues, reviewing, or taking part in discussion — you agree to uphold it. The canonical text is maintained by the CNCF Foundation:

- CNCF Community Code of Conduct: <https://github.com/cncf/foundation/blob/main/code-of-conduct.md>

Until and unless this project is hosted by the CNCF, this Code of Conduct is adopted voluntarily by the project maintainers, and the reporting and enforcement path below applies.

## Scope

This Code of Conduct applies within all project spaces (repository, issues, pull requests, and any official communication channels) and when an individual is representing the project in public spaces.

## Reporting

Report unacceptable behavior to the project maintainers:

- **Contact:** clement.cazaud@outlook.com (interim — a `poesis.cloud` reporting address will replace it). See `MAINTAINERS.md`.

After any CNCF hosting, reports may additionally be made to the CNCF Code of Conduct Committee per the CNCF policy linked above. Reports will be reviewed and investigated promptly and fairly; maintainers are obligated to respect the privacy and security of reporters.

## Enforcement

Maintainers are responsible for clarifying standards and may take appropriate, fair corrective action in response to behavior they deem inappropriate, threatening, offensive, or harmful — including removing, editing, or rejecting contributions and comments, and temporarily or permanently banning any contributor for other behaviors they deem inappropriate.

## Attribution

This Code of Conduct adopts the CNCF Community Code of Conduct, which is itself derived from the Contributor Covenant.
